Powston custom code to work with Flow Power's Flow Home plan.

The Flow Home plan, as of Dec 2025, has a fixed 35/45c export rate between 5:30-7:30pm local time and wholesale linked (PEA adjusted) import rates.

The goal of this code is to add additional functionality on top of Powston Magic/user mode to chage up the battery at the lowest cost, to maximise earnings.
